ポートフォリオ

Webデザイン

制作期間

（個人制作）/ 2020年

使用したソフトウェア

Visual Studio Code (HTML / CSS / JavaScript) / Illustrator / Photoshop

作品説明

このポートフォリオサイトは、授業でJavaScriptは扱ったことがあったけれど、HTMLとCSSに一切触れたことがなかった私が、独学でコーディングまで行った作品です。
昨今の状況をみて、紙媒体のポートフォリオを製作するよりも、Web媒体で製作する方が良いのではないかと考えポートフォリオ製作を始めました。既存のポートフォリオサイトはシンプルなものが多く、満足できなかったので、それなら自分で作ればいいのではないかと思い、一から全て作りました。

見やすさを第一に。

楽しさを第二に。

今までの作品をまとめていたら、世界観が重要な作品があったので、それらは別途それぞれの作品に合った世界観で表現しました。

このポートフォリオ自体は、レトロポップな雰囲気を目指しました。
