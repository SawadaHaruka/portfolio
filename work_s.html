しんかい2020

ゲームデザイン

制作期間

１ヶ月（授業課題）/ 2020

使用したソフトウェア

Visual Studio Code (JavaScript) / Illustrator / animateCC

作品説明

しんかい2020は、授業の課題で作ったスマートフォン用のゲームです。
htmlやcssなど存在すら知らなかった中、ほぼ初めてJavaScriptに触れました。canvasタグ内のJavaScriptのみに取り組みました

深海生物が大好きだったので、「しんかい2020」という深海2000メートル辺りを潜る、架空の潜水艦を操作して、深海生物を採取するというゲームを作りました。

トップページとプレイ方法の説明ページのアニメーションや、深海生物の動き、時間制限に力を入れて作りました。その他深海生物の個体数により点数を変えることや、深海生物の向きを変える等の工夫をしました。
ライブラリはcreatejsを使用しました。スタートボタンから遊べますのでぜひ飛んでみてください。
